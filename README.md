# UniTools Wallet

UniTools Wallet is a demo decentralized application built for the Unicity Testnet.

## Features

- Wallet connection
- Balance viewer
- Transaction history
- Block explorer
- Testnet faucet support

## Purpose

This project demonstrates a simple wallet experience for developers and users exploring the Unicity ecosystem.

## Tech Stack

- HTML
- CSS
- JavaScript

## Status

Demo project for the Unicity Developer Portal.
